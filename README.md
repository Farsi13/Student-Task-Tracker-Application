# Student Task Tracker

A Python application to organize and manage daily tasks using **Object-Oriented Programming (OOP)**. Users can **add, view, update, and delete tasks**, all stored in a JSON file for persistence.

---

## Features

- Add new tasks with a title and description.
- View all tasks with creation timestamp.
- Update task title or description.
- Delete tasks with confirmation.
- Tasks are automatically loaded from and saved to a JSON file.
- Simple numbering for easy task management.

---

## Technologies

- Python 3
- Classes: `Task` and `TaskManager`
- JSON for storing tasks
- `datetime` module for timestamps
- `random` module for unique IDs
- File handling and error handling
